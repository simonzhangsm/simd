template<unsigned N, class E1>
Ret reduce_or( const Vec1<N,E1>& a );

Computes bitwise OR of all elements in the vector. The implementation behaves as if the following set of overloads is provided:

Ret Vec1
int8_t int8
uint8_t uint8
int16_t int16
uint16_t uint16
int32_t int32
uint32_t uint32
int64_t int64
uint64_t uint64
